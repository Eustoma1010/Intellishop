📊 FLOW DIAGRAM

REGISTRATION                        LOGIN
     │                               │
     ├─ 1A: Local Form               ├─ 2A: Local (Email+Pass)
     │   └─ Create User              │   └─ Verify Password
     │   └─ Hash Password            │   └─ Check Social-only
     │   └─ Auto Login               │   └─ JWT Tokens
     │
     └─ 1B: Google                   └─ 2B: Google
         ├─ Email exists?                ├─ Email exists?
         │   NO → SCENARIO 2             │   NO → SCENARIO 2
         │        (Requires Info)        │        (Requires Info)
         │        └─ Collect Phone       │
         │        └─ Collect Address     │   YES → Check Link
         │        └─ Create User         │        ├─ Linked
         │        └─ Link Google         │        │  └─ Login
         │        └─ JWT Tokens          │        │
         │        └─ Auto Login          │        └─ Not Linked
         │   YES → See 2B                │           (Local User)
         │                               │           ├─ Link Auto
         │                               │           ├─ Login
         │                               │           └─ Message
