🔄 THE 4 AUTHENTICATION FLOWS IMPLEMENTED

✅ LUỒNG 1A: Đăng ký Local (Form)
   Email + Password + Name + Phone + Address
   → Validate → Create User (hash password) → Auto-login

✅ LUỒNG 1B: Đăng ký Google (New User)
   Google Account → Email not exist → Kịch bản 2
   → Collect Phone + Address → Create User + Link Google → Auto-login

✅ LUỒNG 2A: Đăng nhập Local (Email + Password)
   Email + Password → Validate → JWT tokens → Auto-login

✅ LUỒNG 2B: Đăng nhập Google (Smart Merge)
   Google Account → Email exists?
   ├─ Linked → Login
   └─ Local User → Auto-link + Login
      (Same cart, orders, history preserved)
