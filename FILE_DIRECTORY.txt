🚀 QUICK START GUIDE
═══════════════════

If you only have 10 minutes:

1. Read: 00_START_HERE_GOOGLE_OAUTH.txt (this file!)
2. Go to: https://console.cloud.google.com/
3. Create: OAuth 2.0 credentials
4. Copy: Client ID & Secret
5. Edit: intelishop_backend/.env
   Add: GOOGLE_OAUTH_CLIENT_ID=your-id
   Add: GOOGLE_OAUTH_CLIENT_SECRET=your-secret
6. Edit: intelishop_frontend/index.html
   Replace: YOUR_GOOGLE_CLIENT_ID_HERE
7. Setup: Django Admin Social Application
8. Test: Run frontend & backend, click Google button
