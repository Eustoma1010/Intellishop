🛠️ TROUBLESHOOTING

If PostgreSQL migration fails:

  1. Check DATABASE_URL in .env
  2. Test connection: psql $DATABASE_URL -c "\l"
  3. Run migrations with verbose: python manage.py migrate --verbosity=3
  4. Check logs for specific errors

If data won't load:

  1. Verify JSON format: python -m json.tool intelishop_export.json
  2. Try loading subset first: python manage.py loaddata core
  3. Check fixture order (dependencies)
  4. Run: python manage.py loaddata --verbosity=3

If admin panel doesn't load:

  1. Check migrations: python manage.py showmigrations
  2. Verify user accounts: User.objects.all()
  3. Check CSRF settings
  4. Review browser console for errors
